# FreshCart — An (Almost) Fully-Functional E-Commerce Project

Welcome to FreshCart — an e-commerce project built with a mix of curiosity, persistence, and a fair amount of debugging.

## What’s Implemented So Far

Here’s what’s currently working:

* Authentication and token handling using NextAuth

* Server-Side Rendering (SSR) with Next.js for the Home and Products pages to improve SEO

* Payment integration using Stripe:

  * Users can pay with Visa
  * Or choose Cash on Delivery
  * Note: The payment data is fake, but just in case, avoid using real card details. If anything unexpected happens, refunds are unfortunately not part of the feature set

* Core shopping features:

  * Add to Cart
  * Add to Wishlist
  * Dedicated pages for both, fully routed and responsive

* Proper handling of empty states for cart and wishlist

## Work in Progress
